## 摘要

本方案评估 TTS/STT 技术集成路径，实现飞书机器人的语音交互能力。参考 airi 项目的实时语音聊天架构，结合 OpenClaw 现有 tts 工具，设计分阶段实施方案。

**核心目标：**
- 集成 TTS（文本转语音）能力
- 集成 STT（语音转文本）能力
- 实现飞书语音消息收发
- 支持实时语音对话（低延迟）

**预期成果：**
- 飞书语音消息发送/接收
- 语音指令识别
- 实时语音对话原型

#### 2.2 模块设计

**模块 1: STT 模块 (`src/voice/stt.ts`)**
```typescript
interface STTModule {
  // 语音文件转文本
  transcribe(fileKey: string): Promise<string>;
  
  // 流式语音转文本（实时对话）
  transcribeStream(audioChunks: Buffer[]): AsyncIterable<string>;
  
  // 语言检测
  detectLanguage(audio: Buffer): Promise<string>;
}
```

**模块 2: TTS 模块 (`src/voice/tts.ts`)**
```typescript
interface TTSModule {
  // 文本转语音
  synthesize(text: string, options?: TTSOptions): Promise<Buffer>;
  
  // 流式合成（低延迟）
  synthesizeStream(text: string): AsyncIterable<Buffer>;
  
  // 语音配置
  setVoice(voiceId: string): void;
  setSpeed(speed: number): void;
}
```

**模块 3: 语音消息管理器 (`src/voice/manager.ts`)**
```typescript
interface VoiceManager {
  // 接收语音消息
  onVoiceMessage(event: FeishuVoiceEvent): Promise<void>;
  
  // 发送语音回复
  sendVoiceReply(chatId: string, text: string): Promise<void>;
  
  // 实时对话模式
  startRealtimeChat(sessionId: string): Promise<void>;
}
```

#### 4.3 内存通道分配

| 用途 | 比例 | 说明 |
