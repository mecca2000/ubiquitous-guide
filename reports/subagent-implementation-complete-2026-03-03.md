## 🔧 技术细节

### 分类规则

**QUICK（立即回复）：**
- 问候：你好、hello、在吗
- 状态查询：报告状态、当前模型
- 短消息：<6 字
- 确认类：是否、能不能

**TASK（子代理处理）：**
- 文件操作：创建、写入、修改、删除
- 搜索调研：搜索、查找、分析
- 复杂任务：实施、执行、完成
- 下载配置：下载、安装、配置

**NORMAL（正常处理）：**
- 其他普通对话

### 子代理配置

```javascript
{
  label: 'task-worker',
  cleanup: 'delete',      // 完成后自动清理
  timeoutSeconds: 300,    // 5 分钟超时
  capabilities: {
    fileRead: true,
    fileWrite: true,
    network: true,
    shell: false,         // 安全优先
    sensitive: false
  }
}
```
