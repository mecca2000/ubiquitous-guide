## 一、问题诊断

| 问题 | 描述 | 根本原因 |

| **1. 多消息队列未实现** | "你好"后面的消息都没有回复 | bot.ts 入队后直接 return，没有继续执行回复逻辑 |

### 修复 2: 多消息智能整合 ✅

**新增文件：** `queue/message-aggregator.ts` (116 行)

**功能：**
- 3 秒窗口期聚合同一用户的连续消息
- 最多聚合 10 条消息
- 自动合并为一条处理

**核心代码：**
```typescript
export const AGGREGATION_CONFIG = {
  windowMS: 3000,      // 3 秒窗口期
  maxMessages: 10,     // 最多聚合 10 条
};

export function addMessageToAggregator(event: FeishuMessageEvent): {
  shouldProcess: boolean;
  aggregated?: AggregatedMessage;
}
```

**使用示例：**
```
用户：帮我查一下
用户：今天沈阳的
用户：天气
→ 聚合为："帮我查一下 今天沈阳的 天气"
→ 作为一条消息处理
```

**修改文件：** `reply-dispatcher.ts`

**修复前：**
```typescript

// 所有消息都启用流式输出
const useStreaming = shouldUseStreaming(text) || text.length > textChunkLimit;
```
