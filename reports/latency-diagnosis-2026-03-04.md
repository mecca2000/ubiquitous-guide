## 根本原因

### 主要瓶颈：模型 API 响应慢

**证据：**
1. typing indicator 每 6 秒一次（飞书超时重传机制）
2. 工具调用 2.8 分钟才执行
3. Gateway 和 Agent 处理都很快（<10ms）

**可能原因：**
1. **阿里云 API 慢** - qwen3.5-plus 响应慢
2. **网络延迟** - 服务器到阿里云 API 的网络
3. **模型负载高** - API 端点拥堵
4. **Token 太多** - 上下文过长导致推理慢
