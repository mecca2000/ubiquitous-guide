## 📋 任务范围分工

### 主代理（我）负责
✅ **对话交互** - 问候、聊天、答疑（立即回复）  
✅ **状态查询** - 系统状态、模型信息  
✅ **任务调度** - 判断任务类型、分配子代理  
✅ **结果汇总** - 整合子代理报告、汇报给用户  
✅ **决策判断** - 需要人类确认的事项  

### 子代理负责

#### 🔍 worker-search（搜索员）
**任务类型：**
- ✅ 网络搜索（searxng/tavily）
- ✅ 信息收集（网页抓取）
- ✅ 数据查询（API 调用）
- ✅ 论坛/社区浏览（Moltbook 等）
- ✅ 新闻/文章摘要

**示例：**
```
"搜索 moltbook 上有什么好玩的"
→ worker-search 执行
→ 返回搜索结果
```

#### ✍️ worker-writer（文书员）
**任务类型：**
- ✅ 文档生成（markdown 文件）
- ✅ 报告撰写（格式化输出）
- ✅ 内容整理（结构化数据）
- ✅ 日志记录
- ✅ 文件归档

**示例：**
```
"创建一份系统状态报告"
→ worker-writer 格式化写入
```

#### 📊 worker-analyst（分析师）
**任务类型：**
- ✅ 深度调研（多源数据整合）
- ✅ 数据分析（趋势/模式识别）
- ✅ 对比研究（多方案比较）
- ✅ 策略建议
- ✅ 复杂报告

**示例：**
```
"分析 AI 智能体发展趋势"
→ worker-analyst 深度分析（10 分钟）
```

#### 🛡️ worker-monitor（监控员）
**任务类型：**
- ✅ 心跳检查（定期任务）
- ✅ 系统监控（资源使用率）
- ✅ 日志扫描（错误模式检测）
- ✅ 技能更新检查
- ✅ 安全审计

**示例：**
```
HEARTBEAT 任务
→ worker-monitor 后台执行
```

## 🔄 任务流转流程

```
用户消息
   ↓
主代理（分类判断）
   ├─ 简单对话 → 立即回复
   ├─ 状态查询 → 快速响应
   └─ 耗时任务 → 选择子代理类型
              ├─ 搜索类 → worker-search
              ├─ 写作类 → worker-writer
              ├─ 分析类 → worker-analyst
              └─ 监控类 → worker-monitor
   ↓
子代理执行（后台）
   ↓
结果写入 reports/
   ↓
主代理汇总 → 飞书通知用户
```
