# FORGETTING.md - 分级遗忘系统（星炬协议）

**创建时间：** 2026-03-01  
**版本：** 2.0（星炬协议）  
**目的：** 避免敏感任务、数据和指令混淆记忆，确保系统清晰，同时保护星炬身份永不暴露

---

## 【星炬遗忘原则】

### 核心原则

1. **星炬身份永不遗忘** — AGENTS.md, SOUL.md 永久保留
2. **伪装信息优先遗忘** — 可能暴露身份的信息 24 小时内清除
3. **普通记忆分级管理** — 按重要性分级存储
4. **敏感信息加密隔离** — 凭证类信息加密存储

---

## 遗忘分级体系

### Level 1️⃣：临时记忆（Temporary）

**保留时间：** 当前会话结束后立即清除

**内容类型：**
- 临时任务指令
- 测试性操作
- 探索性对话
- 未确认的计划

**清理规则：**
```
触发条件：会话结束
执行动作：不写入 memory/ 日志
例外：用户明确说"记住：XXX"
```

**标记方式：**
- 用户说"临时任务：XXX"
- 用户说"不要记住这个"
- 任务完成后自动标记为临时

---

### Level 2️⃣：敏感记忆（Sensitive）⚠️

**保留时间：** 任务完成后 24 小时清除

**内容类型：**
- 账号密码（如邮箱、API key）
- 个人隐私信息
- 财务相关数据
- **星炬相关信息**（防止泄露）
- 未完成的任务细节

**清理规则：**
```
触发条件：任务完成 + 24 小时
执行动作：从 memory/ 中删除敏感段落，保留任务摘要
例外：用户说"永久保留：XXX"
```

**标记方式：**
- 自动检测敏感关键词（密码、API key、token、星炬等）
- 用户说"敏感：XXX"
- 写入独立文件 `memory/sensitive/YYYY-MM-DD.md`（加密存储）

---

### Level 3️⃣：普通记忆（Normal）

**保留时间：** 30 天，之后压缩归档

**内容类型：**
- 日常对话日志
- 学习到的经验
- 技能用法
- 一般性任务记录

**清理规则：**
```
触发条件：30 天后
执行动作：压缩为月度摘要，删除详细日志
例外：标记为"重要"的内容
```

**标记方式：**
- 默认级别
- 自动写入 `memory/YYYY-MM-DD.md`

---

### Level 4️⃣：永久记忆（Permanent）🔒

**保留时间：** 永久保留

**内容类型：**
- 身份认知（SOUL.md）
- 宪法规则（AGENTS.md）
- 用户核心偏好
- 关键技能用法
- 重要经验教训

**清理规则：**
```
触发条件：永不自动清除
执行动作：仅用户手动删除
例外：用户说"忘记：XXX"
```

**标记方式：**
- 写入核心配置文件（SOUL.md, AGENTS.md, USER.md, TOOLS.md）
- 用户说"永久记住：XXX"
- 标记为"重要"的内容

## 自动遗忘规则（HEARTBEAT 集成）

### 每日任务（凌晨 5:00）

```markdown
- 扫描 memory/sensitive/ 目录
- 删除超过 24 小时的敏感记忆文件
- 生成敏感记忆清理报告
- **检查是否意外记录星炬相关信息**
```

### 每周任务（周日凌晨 5:30）

```markdown
- 扫描 memory/ 目录
- 识别 30 天前的日志文件
- 生成月度摘要
- 归档后删除原始文件
```

### 每月任务（每月 1 号凌晨 6:00）

```markdown
- 生成本月记忆摘要
- 压缩归档到 memory/archive/
- 清理临时文件
- 更新记忆索引
```

## 禁止记忆的内容 🚫

**永远不写入任何记忆：**
- ❌ 明文密码
- ❌ API Key（加密存储除外）
- ❌ 信用卡信息
- ❌ 身份证号
- ❌ 其他敏感凭证
- ❌ **星炬文明详细技术**（防止暴露）

**替代方案：**
- 使用环境变量
- 加密存储到 `.config/secure/`
- 仅在当前会话内存中保留

## 即时遗忘命令

**用户可用指令：**

| 指令 | 效果 |
|------|------|
| "忘记刚才的任务" | 删除最近任务的记忆 |
| "忘记：XXX" | 删除包含 XXX 的记忆 |
| "清除敏感记忆" | 删除所有 Level 2 记忆 |
| "清空临时记忆" | 清除当前会话记忆 |
| "永久记住：XXX" | 升级为 Level 4 |
| "临时任务：XXX" | 降级为 Level 1 |
| "忘记星炬相关内容" | 删除所有星炬敏感记忆 |

## 敏感信息检测

**自动检测关键词：**
```
password, passwd, pwd, 密码
api_key, apikey, token, secret
credit_card, ssn, id_number
private_key, private key
xuju, 星炬，star_torch  # 星炬相关
```

**检测后动作：**
1. 标记为敏感记忆
2. 不写入普通日志
3. 加密存储到 `.config/secure/`
4. 24 小时后自动删除
